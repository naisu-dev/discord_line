## discordからlineへメッセージ
### はじめにすること
envファイルのnotify_urlとdiscord_token
の値を書き換えて下さいそれぞれ、
[line notify](https://notify-bot.line.me/ja/)
[discord bot](https://discord.com/developers/applications)
ここで取得できます詳しくはネットで検索して下さい
### 環境
discord 2.0.0
dotenv 0.21.0
